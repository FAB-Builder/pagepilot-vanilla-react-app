# pagepilot-vanilla-react-app

Vite + React starter for the PagePilot app.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## GitHub Pages

This project is configured to build with the correct GitHub Pages base path for the `pagepilot-vanilla-react-app` repository.
The workflow in `.github/workflows/deploy.yml` publishes the `dist` folder to the `gh-pages` branch on every push to `main`.
In repository settings, set GitHub Pages to deploy from the `gh-pages` branch at the root folder.
